# Contributing to PIXN

Thank you for your interest in contributing.

This project is currently in a **Pre-MVP stage**.
The focus is on validating architecture and execution boundaries.

---

## How to Contribute

- Open an issue before submitting major changes
- Keep pull requests small and focused
- Follow existing code structure and documentation

---

## Scope Limitations

Please note:
- No AI model implementations are accepted at this stage
- No production-ready features are expected
- Breaking changes are allowed

---

## Philosophy

Execution before intelligence.
Working code before narratives.
